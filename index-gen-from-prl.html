What this does

  • Each selected .prl file will be traversed, and steps with telemetry in them as data references will be extracted. Steps without telemetry will be ignored.
  • For each procedure, a Display Layout will be genned that includes all its telemetry points as alphanumerics. Each alphanumeric will have a text label, based on the telemetry path. Labels will also be added for procedure step numbers.
  • For each procedure, a Stacked Plot will be genned that includes all its telemetry points.
  • A single Tabs view will be created that includes all the generated Display Layouts.
  • Folders that hold the layouts and plots.
  • A single folder that holds all of the above, named using your input Root Name.

How to use

  1. Set parameters below. Label Width is recommended to be 220.
  2. Press the "Choose File" button and select one or more Pride .prl files. A JSON file will immediately be generated.
  3. Press "DOWNLOAD" to download the generated JSON file.
  4. Import the generated file into your Open MCT environment using the "Import from JSON" action.
