Generate Open MCT Stacked Views from CSV

Settings

Output Base Name

Inputs

Telemetry CSVs

Output

What this does

  • Allows the input of multiple CSV files to create stacked views (Stacked Plots and Flexible Layouts), one for each CSV file that is input. Each stacked view is named from its input CSV file name.
  • Stacked views can contain nested Overlay Plots, or parameters on their own. Overlay Plots can be shared amongst multiple stacked views by including the same Overlay Plot name in multiple sheets.
  • Folders will be created to hold Stacked Plots, Flexible Layouts and Overlay Plots respectively.
